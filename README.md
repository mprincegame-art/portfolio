# 🌐 PJr — Personal Portfolio

A responsive, mobile-first personal portfolio built with pure HTML, CSS and JavaScript. Designed with a dark theme, smooth animations, and a clean professional aesthetic.

---

## ✨ Features

- 📱 Mobile-first & fully responsive design
- 🌙 Dark mode aesthetic with green accent
- 🎨 Smooth scroll reveal animations
- 📋 Bottom sheet navigation with swipe to close (mobile)
- 🖱️ Custom cursor with follower effect (desktop)
- 🔝 Scroll to top button (centered)
- 📬 Contact form with response time indicator
- 🗂️ Separate projects page with category filter
- 🔍 SEO optimized — meta description & Open Graph tags
- 🐦 Twitter/X Card support for social sharing
- 🔖 Custom favicon for all devices
- 🚫 Custom 404 error page
- ⚡ Pure HTML, CSS & JS — no frameworks

## 📁 Project Structure

```
portfolio/
├── index.html            # Main page
├── projects.html         # All projects page
├── 404.html              # Custom error page
├── style.css             # All styles & animations
├── script.js             # Main interactivity & logic
├── projects.js           # Projects filter logic
├── manifest.json         # Web app manifest
├── favicon.ico           # Favicon
├── favicon-16x16.png
├── favicon-32x32.png
├── apple-icon-*.png      # Apple touch icons
├── android-icon-*.png    # Android icons
└── assets/
    ├── photo.png         # Profile photo
    └── cv.pdf            # Downloadable CV
```

## 📄 Pages

| Page | Description |
|------|-------------|
| `index.html` | Hero, About, Skills, Projects preview, Contact |
| `projects.html` | All projects with category filter (All / Web / Data / Python) |
| `404.html` | Custom error page with navigation back to home |

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/mprincegame-art/portfolio.git
```

2. Open `index.html` in your browser — no build tools needed.
